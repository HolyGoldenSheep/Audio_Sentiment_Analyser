# Audio Sentiment Analysis API

API FastAPI permettant de détecter les émotions à partir de fichiers audio.

## Technologies
- FastAPI
- MongoDB (Motor)
- JWT Authentication
- TensorFlow / Wav2Vec2
- Pytest

## Installation

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
